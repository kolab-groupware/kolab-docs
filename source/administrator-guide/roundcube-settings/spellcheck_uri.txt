.. _admin_roundcube-settings_spellcheck_uri:

``spellcheck_uri``
==================

For locally installed Nox Spell Server or After the Deadline services, please specify the URI to call it.
Get Nox Spell Server from http://orangoo.com/labs/?page_id=72 or the After the Deadline package from http://www.afterthedeadline.com.
Leave empty to use the public API of service.afterthedeadline.com.
