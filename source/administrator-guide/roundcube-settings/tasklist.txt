``tasklist_driver``
===================

Backend type (``database``, ``kolab``). Default is ``kolab``.

``tasklist_sort_col``
=====================

Default sorting order of tasks listing (``auto``, ``datetime``, ``startdatetime``, ``flagged``, ``complete``, ``changed``).

``tasklist_sort_order``
=======================

Default sorting order for tasks listing (``asc`` or ``desc``). Default is ``asc``.
